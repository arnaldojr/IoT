## Como clonar o repositório usuários Linux

``` bash
$ # Clone o repositório
$ git clone https://github.com/arnaldojr/IoT/
$ cd IoT
$ # instale o Simulide
$ ./guias/install_simulide.sh
$ # instale as dependencias python
$ pip install -r requirements.txt --user
```
